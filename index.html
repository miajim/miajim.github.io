Mia Jimenez

My interest in software stems from my technical background in structural engineering. After years of using VBA and MATLAB to solve structural engineering problems, I realized that my interest lay more in understanding the nuances of existing programming paradigms and using code as a tool to solve a wider array of problems. I recently built Artemis, an open-source, serverless framework for API load testing built using Node.js, AWS Timestream, k6, Grafana, and more. To find out more about Artemis, read the case study linked below.

Artemis

Artemis is an open-source, serverless framework for scalable API load testing. It is an easily deployable, cloud-based load testing framework that provides data retention via AWS Timestream and near real-time result visualization via a customizable Grafana dashboard.
